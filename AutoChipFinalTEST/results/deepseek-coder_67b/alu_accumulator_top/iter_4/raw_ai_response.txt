```verilog
module alu_accumulator_top(
    input clk, rst, load_acc,
    input [7:0] data_in,
    input [2:0] op,
    output wire [7:0] acc_out,
    output wire zero
);
    
    // Internal wires and registers
    wire [7:0] alu_result;
    reg [7:0] acc = 8'h00;
  
    // Instantiate 'alu_8bit'
    alu_8bit u_alu(
        .A(acc), 
        .B(data_in), 
        .op(op), 
        .result(alu_result), 
        .zero(zero)
    );
  
    // Accumulator register
    always @(posedge clk or posedge rst) begin
        if (rst) acc <= 8'h00;
        else if (load_acc) acc <= data_in;
        else acc <= alu_result;
    end
  
    assign acc_out = acc;
    
endmodule
```
This Verilog code creates a module named 'alu_accumulator_top'. It takes inputs clk, rst, load_acc, data_in and op. The output is acc_out and zero. 

The internal wires and registers are declared as per the problem statement. An instance of 'alu_8bit' is created with named port connections. 

An always block is used to describe an accumulator register which updates its value on every positive edge of clock or reset. The updated value depends on whether load_acc is high, in which case it takes data_in as input, otherwise it takes alu_result as input. Finally, the current value of acc is assigned to acc_out.
