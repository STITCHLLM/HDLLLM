```verilog
module param_register_file #(parameter N = 4, parameter W = 8) (input clk, wr_en, input [1:0] wr_addr, rd_addr, input [W-1:0] wr_data, output wire [W-1:0] rd_data);
  reg  [W-1:0] mem [0:N-1];
  
  always @(posedge clk) begin
    if (wr_en) begin
      mem[wr_addr] <= wr_data;
    end
  end
  
  assign rd_data = mem[rd_addr];
endmodule
```
This Verilog code creates a parameterized register file with N as the number of registers and W as the width of each register. The write operation is synchronous, meaning it happens on the rising edge of the clock signal. The read operation is combinational, meaning its output depends only on the current state of the inputs and does not change over time.
