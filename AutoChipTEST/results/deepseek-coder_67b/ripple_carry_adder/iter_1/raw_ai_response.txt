Based on your requirements, here is a Verilog module that implements a ripple carry adder using four full adders. The `full_adder` module is assumed to be defined elsewhere in the code.

```verilog
module ripple_carry_adder(input [3:0] A, B, input cin, output [3:0] Sum, output cout);
    wire c1, c2, c3; // Carries from full adders
    
    // Instantiate four full adders
    full_adder fa0(.a(A[0]), .b(B[0]), .cin(cin), .sum(Sum[0]), .cout(c1));
    full_adder fa1(.a(A[1]), .b(B[1]), .cin(c1), .sum(Sum[1]), .cout(c2));
    full_adder fa2(.a(A[2]), .b(B[2]), .cin(c2), .sum(Sum[2]), .cout(c3));
    full_adder fa3(.a(A[3]), .b(B[3]), .cin(c3), .sum(Sum[3]), .cout(cout));
endmodule
```
This module takes two 4-bit inputs `A` and `B`, a carry in `cin`. It outputs four bits `Sum` and a carry out `cout`. The sum is calculated by chaining together four full adders (fa0 through fa3). Each full adder's carry output feeds into the next one, creating the ripple effect.
